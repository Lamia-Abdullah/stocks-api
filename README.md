
# Tracker Marketing Stocks “ TMS” API 📈

### Features 🚀

# User
1. LogIn
2. SignUp
3. Search for stocks
4. view all stocks
5. view informations for stock
6. Buy stock
7. Sell stock

# Admin
1. signIn
2. SignIn
3. View all users
4. delete user

## Stack
1. Runtime Environment: `Node.js`
2. Language: `TypeScript`
3. Web Server: `Fastify`
4. ORM: `Prisma`
5. DB: `MongoDB`
